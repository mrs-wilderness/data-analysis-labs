# Data Analysis Labs

A collection of data analysis coursework notebooks (Python, R + Jupyter).
Each lab is self-contained, with its own dataset and reproducible code.

## Lab 01 — Exploratory Data Analysis (UCI Adult)

Exploratory analysis of the UCI Adult Income dataset, comparing weighted vs. unweighted distributions and examining income patterns across demographic groups.

## Lab 02 — Visual Data Analysis (Cardiovascular Disease)

An assignment-driven visual analysis of the CVD dataset, combining statistical graphics with a custom Shakespeare-inspired colour palette to highlight feature patterns and relationships.

## Lab 03 — Introductory EDA in R (Iris & Mtcars)
A concise introduction to exploratory data analysis in R, demonstrating basic inspection, visualization, and statistical testing using two classic built-in datasets (Iris and Mtcars).
